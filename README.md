Driver Drowsiness Detection and Alert System

Overview

This project is a real-time driver monitoring system that detects drowsiness using computer vision. It continuously analyzes facial landmarks and eye states through a webcam. When signs of sleepiness are detected, the system immediately triggers an alarm to alert the driver.

⸻

Features
	•	Real-time face detection using webcam
	•	Eye state monitoring (open/closed detection)
	•	Drowsiness detection based on eye closure duration
	•	Alarm alert system to wake the driver
	•	Lightweight and fast detection system

⸻

Technologies Used
	•	Python
	•	OpenCV
	•	MediaPipe
	•	NumPy
	•	Pygame
